# AI Spend Audit Platform

A full-stack AI spend optimization platform built using Next.js, React, TypeScript, and Tailwind CSS.

## Features

- AI spend analysis
- Smart recommendations
- Savings dashboard
- Audit history
- Responsive UI
- AI tool recommendations

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel Deployment
